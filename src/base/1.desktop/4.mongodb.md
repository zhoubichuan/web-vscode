# 四.搜索

- 单行注释

```sh
//.+?(?=["$\r\n]
```

- 多行注释

```sh
\/\*([\w\W]+?)\*\/
```

- 空行

```sh
^\s*(?=\r?$)\n
```

- 全部替换

```sh
((\/\*([\w\W]+?)\*\/)|(\/\/.+?(?=["$\r\n]))|(^\s*(?=\r?$)\n))
```
